# travel-chatGPT-app
chat-GPT를 이용해 여행 일정 계획을 도와주는 애플리케이션 프로젝트<br/><br/>
게스트로 입장해도 채팅 기능과 커뮤니티의 내용을 확인 가능하게 작업했습니다.<br/><br/>
회원의 경우 채팅방을 여러개 만들 수 있으며 커뮤니티에 글을 작성할 수 있습니다.
